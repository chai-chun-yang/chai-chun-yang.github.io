Circuit exploration

Topic background

You have a circuit box with $k$ wires up and down.
We named the top row of wire ends $A_ 1,A_ 2,A_ 3…… A_ k$, next row is $B_ 1,B_ 2,B_ 3…… B_ k$.
For each $A_ i (i\in[1, k]) $ has a unique $B_j (j\in[1, k])$ corresponds to it.
You now have a tester that can check whether any two wire ends are connected.
Please use this tester to measure the correspondence between $A$ and $B$.

Title Description

This is a function interaction problem.
Enter only one line, and two numbers represent $k$ and $t$. ($t$ is defined below)
You can call the following functions:

  1. $\text{void}\ \color{blue}{\text{Merge}}(\color{purple}{\text{int }}\color{blue}{\text{x}},\color{purple}{\text{int }}\color{blue}{\text{y}},\color{purple}{\text{bool }}\color{blue}{\text{n}})$
    Requirement: $x\not=y; x,y\in[1,k]$
    Function: connect the $x$ wire head and $y$wire head in line $n$ into a path
  2. $\text{void}\ \color{blue}{\text{Split}}(\color{purple}{\text{int }}\color{blue}{\text{x}},\color{purple}{\text{int }}\color{blue}{\text{y}},\color{purple}{\text{bool }}\color{blue}{\text{n}})$
    Requirement: $x\not=y; x,y\in[1,k]$
    Function: disassemble the $x$ wire head and $y$ wire head in line $n$
  3. $\text{void}\ \color{blue}{\text{Check}}(\color{purple}{\text{int }}\color{blue}{\text{x}},\color{purple}{\text{bool }}\color{blue}{\text{xn}},\color{purple}{\text{int }}\color{blue}{\text{y}},\color{purple}{\text{bool }}\color{blue}{\text{yn}})$
    Requirements: $x, y\in[1, k]$, the number of calls shall not exceed $t$.
    Function: check whether the $x$ root in line $xn$ and the $y$ root in line $yn$ are paths. The Unicom returns $true$, otherwise it returns $false$.

Note: the above $n$, $xn$, $yn$ all represent the line where the wire head is located
$(n=0)\to A,(n=1)\to B$

Input format

See Title Description

Output format

Only one line $k$ number, the $i_{th}$ number $x$ represents $A_i$ and $B_x$ corresponds.
