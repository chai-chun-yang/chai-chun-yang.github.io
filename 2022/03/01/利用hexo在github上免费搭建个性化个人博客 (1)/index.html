    利用hexo在github上免费搭建个性化个人博客

    发表于 2022-03-01

    利用hexo在github上免费搭建

    个性化个人博客

    ===

    First

    在github上创建一个新项目。

    注意

    1.Repository name 中填yourname.github.io (yourname自己填)

    2.选择public（公开）

    Second

    安装 git 和 node.js

    ==

    全部选默认即可

    安装完成后，在控制台中输入

    “node -v”和”git --version”

    如果成功则会显示当前的版本号

    ===

    Third

    打开 Git Bash 输入”ssh-keygen -t rsa -C email”

    email为你注册github所用的邮箱

    接下来输入3次换行

    在C盘下用户文件夹里进入你当前用户

    打开 .ssh 文件夹，找到”id_rsa.pub”

    同时在github项目中找到

    Settings -> Deploy keys -> Add deploy key

    title 填什么都可以，Key 中粘贴

    我们刚刚在 id_rsa.pub 文件中的内容

    保存后，在 Git Bash 中输入”ssh -T git@github.com“

    看到”You’ve successfully authenticated”即为成功

    Forth

    在你所选的盘根目录下创建文件夹，名字自己定

    在其目录控制台中输入”npm insatll hexo -g”

    完成后输入 “hexo -v” 检查安装是否成功

    在此处用Git Bash

    依次输入

    “hexo init”

    “npm install”

    “hexo g”

    “hexo s”

    如果显示

    即为成功，可以在 localhost:4000 本地查看

    注：添加文章可以在\source_posts 中添加

    Fifth

    打开_config.yml文件，将deploy字段改为如下内容

    git@github.com:someurl/someurl.github.io.git

    是博客仓库的地址，获取方式是登录GitHub，

    找到自己的博客的仓库

    便可将本地文件上传至github

    然后进入setting 中的Pages

    Source选择master branch

    这意味着我们把我们的博客

    以及博客配置生成所需要的相关文件

    都放到我们这个仓库的主干中。

    不要点击(Choose a theme)

    Finally

    在 yourname.github.io 上即可查看博客

    完结撒花
